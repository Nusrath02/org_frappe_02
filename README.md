# ITC Org Frappe

An org chart app for Frappe / ERPNext built by ITChamps.

## Features

- **Org Employee** doctype to manage employees with hierarchy
- **Org Chart** page with an interactive visual tree
- Color-coded nodes (Blue, Green, Red, Purple, Orange, Teal)
- Click any node to open the employee form

## Installation

```bash
bench get-app https://github.com/Nusrath02/org_frappe_02
bench --site [your-site] install-app itc_org_frappe
bench --site [your-site] migrate
```
